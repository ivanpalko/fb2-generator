# FB2 Генератор сборников

Генератор FB2 книг-сборников на PHP без базы данных. Программа позволяет создавать книги с несколькими авторами, добавлять тексты, обложки и компилировать в формат FB2.

## Возможности

- 📚 Создание книг-сборников с несколькими авторами
- ✏️ Добавление текстов произведений (поддержка Word и разных кодировок)
- 🖼️ Загрузка и автоматическое сжатие обложек
- 📝 Поддержка аннотаций и подзаголовков
- 🎨 Настройка форматирования через конфиг
- 📖 Разделение на прозу и стихи
- 💾 Хранение данных в файловой структуре (без БД)

## Требования

- PHP 7.4 или выше
- GD библиотека для работы с изображениями
- mbstring расширение
- DOM расширение
